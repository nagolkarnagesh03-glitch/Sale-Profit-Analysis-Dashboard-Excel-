# 📊 Sales & Profit Analysis Dashboard (Excel)

## 📌 Project Overview

This project focuses on analyzing sales performance, profitability, and customer trends using Microsoft Excel. The dashboard provides interactive insights to help businesses make data-driven decisions.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify top-performing regions, categories, and customers
* Track monthly sales trends
* Evaluate delivery performance and operational efficiency

---

## 📂 Dataset Details

The dataset includes the following key fields:

* Order ID, Order Date, Ship Date
* Customer ID, Customer Name, Segment
* Region, State, City
* Category, Sub-Category, Product Name
* Sales, Quantity, Discount, Profit

---

## ⚙️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Slicers for interactivity
* KPI Metrics (Sales, Profit, Orders, Delivery Days)
* Data Cleaning & Transformation
* Conditional Formatting

---

## 📊 Dashboard Features

* 💰 Total Sales, Profit, and Orders (Dynamic KPIs)
* 📈 Monthly Sales Trend Analysis
* 🌍 Sales Distribution by Region
* 🛒 Sales Distribution by Category
* 🧑‍💼 Top 10 Customers Analysis
* 🚚 Delivery Performance Insights
* 🎛 Interactive Slicers (Region, Category, Segment, Year)

---

## 📌 Key Insights

* West region contributes the highest sales revenue
* Technology category generates maximum profit
* Sales peak observed during year-end months
* Delivery time varies across regions impacting efficiency

---

## 🚀 How to Use

1. Open the Excel file
2. Use slicers (Region, Category, Segment, Year)
3. Analyze KPIs and charts dynamically

---


## 💡 Conclusion

This dashboard helps in understanding business performance, identifying growth opportunities, and improving operational decisions using data analytics.
